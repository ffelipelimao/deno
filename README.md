# Deno Skeleton

### Installation

- Use vscode plugin denoland.vscode-deno (the purple)
- `curl -fsSL https://deno.land/x/install/install.sh | sh`

##### put in .bashrc/.zshrc...

- `export DENO_INSTALL="/home/USER/.deno"`
- `export PATH="$DENO_INSTALL/bin:$PATH"`

### Run

- `deno run --allow-net server.ts`
